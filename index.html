ImageMagick Effects

A curated collection of procedural art generated entirely via ImageMagick CLI scripts.

Witness raw pixels transformed into complex textile weaves, oil canvases, and distressed wall art using pure mathematical operations.

This project serves as a live demonstration of how automated, headless bash scripts can replicate physical media variance with precision.

1. Install ImageMagick

You need the ImageMagick command-line tool installed on your computer.

  • Linux (Ubuntu/Debian): sudo apt-get install imagemagick
  • macOS: brew install imagemagick
  • Windows: Download Installer
2. Run the command

Use the command shown under each effect. Replace input.png with your image path and run it in your terminal.

Note regarding command names:
On Linux/Mac (ImageMagick 6), the command is convert.
On Windows or ImageMagick 7+, the command is magick.
If convert doesn't work for you, try changing it to magick.
